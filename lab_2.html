JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, ​ basado en prototipos, imperativo, débilmente tipado y dinámico. Suele decirse que JavaScript es totalmente distinto a Java; sin embargo, se pueden señalar algunas similitudes: tanto JavaScript como Java son lenguajes de programación orientados a objetos; los dos utilizan conceptos como encapsulación, herencia, polimorfismo y abstracción.
